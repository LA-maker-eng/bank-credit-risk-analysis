# bank‑credit‑risk‑analysis
## 项目简介
本项目基于GitHub开源信用卡审批预测项目进行二次开发。原项目采用国外信用卡数据集，使用GBDT模型搭配Streamlit网页部署；结合银行数字化业务场景，我对项目进行针对性改造。

### 主要工作内容
1. 数据集改造：自行编写代码生成20000条国内银行客户模拟数据，替换原项目海外信用卡数据集；
2. 特征工程：完成缺失值填充、异常值剔除，自主新增资产负债率特征；
3. 模型精简：舍弃原项目复杂的GBDT模型，选用业务可解释性更强的逻辑回归模型；
4. 数据分析：借助Matplotlib、Seaborn可视化分析客户逾期规律，输出客户分层运营建议；
5. 版本管理：采用Git分阶段迭代开发，GitHub可查看每一步修改记录。

## 技术栈
Python、Pandas、NumPy、Scikit‑learn、Matplotlib、Seaborn、Git

## 项目成果
模型准确率达到0.84；得出低收入、频繁借贷客户逾期风险偏高的结论，为银行信贷风控、降低坏账提供参考依据。
